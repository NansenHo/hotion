# Hotion

This is a Notion clone project named Hotion.

## Tech Stack

- TypeScript
- React
- Next.js
- [shadcn-ui](https://www.npmjs.com/package/shadcn-ui)
- Vitest
- TailwindCSS

## :hammer_and_wrench: Quickstart

```bash
# clone the project
git clone https://github.com/NansenHo/hotion.git

# install packages
pnpm install

pnpm dev
```
